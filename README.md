#  Demand Forecasting | Time Series | Machine Learning

##  Project Overview

This project focuses on predicting product demand using Machine Learning on retail inventory data. By analyzing key business factors, the model forecasts future demand to support inventory planning and business decision-making.

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* XGBoost
* Jupyter Notebook

## Dataset

* Source: Kaggle
* Retail Inventory & Demand Dataset

##  Project Workflow

### 1. Data Collection

* Imported the retail inventory dataset from Kaggle.
* Loaded the data into Python for analysis and model development.

### 2. Data Cleaning

* Removed duplicate records.
* Checked for missing values.
* Corrected data types and ensured data consistency.

### 3. Data Preprocessing

* Encoded categorical features using Label Encoding.
* Prepared the dataset for machine learning.
* Split the data into training and testing sets.

### 4. Feature Selection

Selected the most relevant features:

* Price
* Discount
* Inventory Level
* Promotion
* Competitor Pricing
* Category

### 5. Exploratory Data Analysis (EDA)

* Explored feature distributions.
* Identified relationships between variables.
* Visualized demand patterns and business insights.

### 6. Model Training

* Built an XGBoost Regression model.
* Trained the model to forecast product demand.

### 7. Model Evaluation

Evaluated the model using:

* R² Score
* MAE
* MSE
* RMSE

### 8. Demand Prediction

* Predicted demand for new user-provided inputs.
* Generated real-time demand forecasts.

### 9. Data Visualization

Created visualizations including:

* Actual vs Predicted Plot
* Feature Importance Chart
* Residual Plot

##  Project Outcome

Successfully developed a Machine Learning-based demand forecasting model capable of predicting product demand and supporting data-driven inventory management decisions.
